# Parser
---
1. 题目：
构造一个小（Mini）语言的语法分析程序<br/>


2. 要求：
输入属性字文件，输出源程序是否符合语法要求的结果：

>正确——该程序符合语法要求。
>错误——指出错误位置。

如：
输入：`i：=1+；`
输出：表达式错误。

输入：`program exl; begin 1:=1 end.`
输出：该程序是正确的。


• 要求 Mini语言至少包含算术表达式、逻辑表达式、赋值语句、if-then 分支结构、while循环结构等基本控制结构。
